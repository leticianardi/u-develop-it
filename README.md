## MySql

To check out a database, type one of the following command on the mysql shell:
```
SHOW DATABASES;
SHOW SCHEMAS;
```

To select a database, run the following command:
```
SELECT schema_name
```

To run a database:
```
source db/schema.sql
source db/seeds.sql
```
